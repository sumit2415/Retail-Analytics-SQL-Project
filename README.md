# 🛍️ Retail Analytics SQL Project

## 📌 Project Overview

This project demonstrates the use of SQL to analyze retail sales data. It covers the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), product performance analysis, customer analysis, sales trend analysis, and business insights.

The project is built using MySQL Workbench and is designed to showcase practical SQL skills required for Data Analyst roles.

---

## 🎯 Objectives

- Clean and validate raw retail data
- Perform exploratory data analysis (EDA)
- Analyze product performance
- Analyze customer purchasing behavior
- Identify monthly sales trends
- Generate business insights from the data

---

## 🛠️ Tools Used

- MySQL Workbench
- SQL

---

## 📂 Dataset

The project uses three datasets:

- **customer_profiles**
- **product_inventory**
- **sales_transaction**

## 🧹 Data Cleaning

The following data cleaning tasks were performed:

- Renamed imported tables
- Renamed columns
- Checked NULL values
- Identified duplicate records
- Created backup table
- Removed duplicate records
- Validated quantity, price and stock values
- Converted date columns
- Verified referential integrity
